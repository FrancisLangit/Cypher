Cypher

Cipher and decipher text using various algorithms.

Currently released on GitHub as an open-source project under an MIT License.

Frequently Asked Questions

What is Cypher?

Cypher is a Django-based web application written on Python 3. Visitors can make use of its features to encrypt English to various ciphers and vice-versa. It currently showcases ciphering algorithms that can translate to and from the following:

  • Binary
  • Caesar Cipher
  • Morse Code
  • Pig Latin

Can I contribute to the project?

You most definitely can. Please visit the project's repository on GitHub for more details. Within it you can learn more about its contributing guidelines.
